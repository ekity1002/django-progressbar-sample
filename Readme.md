# django-progressbar-sample
Sample of Django displaying progressbar for file upload with celery and redis.

## Environment
* Python 3.9.5
* celery           5.2.7
* celery-progress  0.3
* django 4.2
* django-redis     5.2.0

## Usage
Run following command and connect to http://localhost:8000/upload/ with your browser.
```
docker-compose up -d
```
